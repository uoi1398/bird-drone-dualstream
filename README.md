# Bird-Drone Dual-Stream Recognition

This project explores a brain-inspired dual-stream framework for bird-drone recognition using RGB appearance information and optical flow motion information.

## Project Structure

- `src/`: source code
- `configs/`: experiment configuration files
- `notebooks/`: Colab notebooks
- `docs/`: experiment notes and logs

## Experiments

1. RGB-only appearance model
2. Flow-only motion model
3. RGB+Flow late-fusion model

## Notes

Large files such as videos, model checkpoints, and experiment outputs are not stored in this repository.
